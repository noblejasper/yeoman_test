gulpとかもgruntより何がいいかとか分かってないので、それも明確にしたい

```sh:使ったコマンド
npm install -g yo bower grunt-cli gulp
mkdir my-yo-project
cd my-yo-project
yo webapp
```

```sh:backboneもいれたい
rm -r * .*

yo backbone
```
